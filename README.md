# image-hosting

这是一个用于存储图片的仓库（图床）

## 说明

此仓库用于存储和管理图片文件，通过CDN加速访问。

## 目录结构

```
images/              # 图片文件夹
├── YYYYMMDD/       # 按日期分类
└── ...
slider_screenshots/  # 滑块验证码截图
└── login_key.png   # 登录验证码图片
```

## 访问方式

图片可以通过以下方式访问：

1. **CDN加速（推荐）**：
   ```
   https://cdn.jsdelivr.net/gh/kmxs2547/image-hosting@main/路径/文件名
   ```

2. **GitHub Raw**：
   ```
   https://raw.githubusercontent.com/kmxs2547/image-hosting/main/路径/文件名
   ```
